
Target : 目标对象
SystemInfo ： SystemInfo 域定义了用于查询底层系统信息的方法和事件
Browser ： 浏览器域定义了用于管理浏览器的方法和事件。
BackgroundService ：  定义后台 Web 平台功能的事件。
CacheStorage ： 缓存存储域
CSS ： 此域公开 CSS 的读写操作。所有 CSS 对象（样式表、规则和样式）都有一个关联的属性，id用于后续对相关对象的操作。
每种对象类型都有特定的id结构，不同类型的对象结构不可互换。可以使用 `load` 调用（接受 DOM 节点 ID）加载 CSS 对象。
客户端还可以通过`/ ` 事件get*ForNode()跟踪样式表，并随后使用 `load` 方法加载所需的样式表内容。
styleSheetAddedstyleSheetRemovedgetStyleSheet[Text]()
Debugger ： 调试器域公开了 JavaScript 调试功能。它允许设置和移除断点、单步执行代码、查看堆栈跟踪等。
DOM : 此域公开 DOM 读/写操作。每个 DOM 节点都由其镜像对象表示，该镜像对象具有一个 `<object>` 属性id。此id属性可用于获取节点的附加信息、
将其解析为 JavaScript 对象包装器等等。客户端仅接收其已知的节点的 DOM 事件，这一点至关重要。后端会跟踪已发送给客户端的节点，并且不会重复发
送同一个节点。客户端有责任收集有关已发送给客户端的节点的信息。请注意，iframe所有者元素会返回相应的文档元素作为其子节点。
DOMDebugger : DOM调试允许在特定的DOM操作和事件上设置断点。JavaScript执行会在这些操作处停止，就像设置了常规断点一样
DOMSnapshot : 该域便于获取包含 DOM、布局和样式信息的文档快照。
DOMStorage  : 查询和修改 DOM 存储。
Emulation : 该域名模拟了页面的不同环境。
EventBreakpoints : 事件断点域 允许在 JavaScript 调用的原生代码中发生的操作和事件上设置 JavaScript 断点。断点触发后，
会像触发常规断点一样，通过调试器域进行报告。
Extensions : 定义浏览器扩展的命令和事件。
FedCm : 该域允许与 FedCM 对话框进行交互。
Fetch : 允许客户端使用客户端代码替换浏览器网络层的域。
FileSystem : 文件系统域
HeadlessExperimental : 此域提供仅在无头模式下支持的实验性命令。
HeapProfiler : 堆分析器域
IndexedDB : IndexedDB相关的域
Input ： 输入域
Inspector ： 检查域
IO ： 对 DevTools 生成的流进行输入/输出操作。
LayerTree ： 层树
Log ： 提供对日志条目的访问权限。
Media ： 该域允许对媒体元素进行详细检查。
Memory ： 内存相关
Network ： 网络域允许跟踪页面的网络活动。它公开有关 HTTP、文件、数据和其他请求和响应的信息，包括它们的标头、正文、时间等。
Overlay ： 叠加域 该域提供与在被检查页面上绘制图形相关的各种功能。
Page ： 与被检查页面相关的操作和事件属于页面域。
Performance ： 性能域
PerformanceTimeline ： 按照https://w3c.github.io/performance-timeline/#dom-performanceobserver中的规定，报告性能时间线事件
Preload ： 预加载域
Profiler ： 分析器域
PWA ： 该域允许与浏览器交互以控制 PWA。
Runtime ： 运行时域通过远程求值和镜像对象公开 JavaScript 运行时环境。求值结果以镜像对象的形式返回，该对象公开对象类型、
字符串表示形式和唯一标识符，以便后续引用。原始对象会一直保存在内存中，除非被显式释放，或者与其对象组中的其他对象一起释放。
Security ： 安全域
ServiceWorker ： 服务任务
Storage ： 存储
Tethering ： 域定义了浏览器端口绑定的方法和事件。
Tracing ： 追踪
WebAudio ： 此域名允许查看 Web Audio API。https ://webaudio.github.io/web-audio-api/
WebAuthn ： 该域允许配置虚拟身份验证器来测试 WebAuthn API。

 创建新页面。

参数
url string 页面跳转到的初始 URL。空字符串表示 about:blank。
left  整数  DIP 中的帧左原点（需要 newWindow 为真或无头 shell）。 实验
top  整数  DIP 中的帧顶部原点（需要 newWindow 为真或无头 shell）。 实验

height  整数  DIP 中的帧高度（需要 newWindow 为真或无头外壳）。

browserContextId  浏览器.浏览器上下文ID(Browser.BrowserContextID) 创建页面的浏览器上下文。 实验
enableBeginFrameControl   布尔值  是否通过 DevTools 控制此目标的 BeginFrames（仅限无头 shell，MacOS 尚不支持，默认为 false）。 实验
newWindow  布尔值  是否创建新窗口或选项卡（默认为 false，无头 shell 不支持）。
background  布尔值  是否在后台或前台创建目标（默认为 false，无头 shell 不支持）。
forTab  布尔值  是否创建类型为“制表符”的目标。 实验
hidden  布尔值  是否创建隐藏目标。隐藏目标可通过协议观察，但不显示在标签页 UI 栏中。无法使用 `<Tab>`、`<Tab>` 或 `<Tab>` 创建forTab: true。newWindow: true标签 background: false页的生命周期与会话的生命周期相同。 实验
focus  布尔值  如果指定，该选项用于确定是否应将焦点移至新目标。默认情况下，焦点行为取决于 background 字段的值。例如，background=false 且 focus=false 将导致目标标签页打开，但浏览器窗口保持不变（如果它原本在后台运行，则仍将保持在后台）；而 background=false 且 focus=undefined 将导致窗口获得焦点。不支持同时使用 background: true 和 focus: true，这将导致错误。 实验

返回
targetId  目标ID (TargetID) 打开页面的 ID。
